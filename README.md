# REST Assured API Framework

This project contains API automation scripts developed using REST Assured, Java and TestNG.

## Features
- API Request Validation
- Status Code Verification
- Response Validation
- Reusable Utilities
- Reporting Support

## Tools & Technologies
- REST Assured
- Java
- TestNG
- Maven
- GitHub

## API Scenarios
- GET API Validation
- POST API Validation
- DELETE API Validation
